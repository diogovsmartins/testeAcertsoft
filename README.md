# testeAcertsoftPara rodar o programa eu utilizei o xampp e o mysql para o banco de dadosentão é preciso ter os dois instalados. Pra abrir o site no navegador é preciso seguir os passos a seguir:1-Criar um banco de dados2-Criar um scheema dentro dele chamado db_acertsoft3-Criar uma table chamada users com 3 colunas chamadas, id, email e password. É importante que a coluna id seja auto incrementada e tenha a propriedade unique marcada e not null, as outras duas podem ser apenas not null.4-Baixar os arquivos do git e colocar a pasta com eles dentro da pasta htdocs dentro dos arquivos do xampp.5- Abrir o xampp, iniciar o mysql e o apache por lá pra rodar no browser6-Abrir a pasta com os arquivos em algum editor de código e mudar no arquivo config.php  a porta do ip, o user e a senha nas variaveis.7- Abrir o navegador e digitar localhost/ nome da pasta que está no htdocs com os arquivos.
