# :Teste Acertsoft

##  :mega:  Para rodar o programa eu utilizei o xampp e o mysql para o banco de dados então é preciso ter os dois instalados. Pra abrir o site no navegador é preciso seguir os passos a seguir:

* ### :one: -Criar um banco de dados

* ### :two: -Criar um scheema dentro dele chamado db_acertsoft

* ### :three: -Criar uma table chamada users com 3 colunas chamadas, id, email e password. É importante que a coluna id seja auto incrementada e tenha a propriedade unique marcada e not null, as outras duas podem ser apenas not null.

* ### :four:  -Baixar os arquivos do git e colocar a pasta com eles dentro da pasta htdocs dentro dos arquivos do xampp.

* ### :five:  - Abrir o xampp, iniciar o mysql e o apache por lá pra rodar no browser

* ### :six:  -Abrir a pasta com os arquivos em algum editor de código e mudar no arquivo config.php  a porta do ip, o user e a senha nas variaveis.

* ### :seven: - A brir o navegador e digitar localhost/ nome da pasta que está no htdocs com os arquivos.
